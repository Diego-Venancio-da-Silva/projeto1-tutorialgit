Comandos git:
1 - git init
2 - git status
3 - git add "arquivo" << adiciona o arquivo
4 - git add . << adiciona tudo
5 - git commit -m "commit inicial" << 'm' significa mensagem, primeira versão de um arquivo
6 - git push <name>  << coloca o arquivo na nuvem do github
7 - git reflog << ver as versões dos arquivos
8 - git reset --hard <id da versão do arquivo>
9 - git branch << ver as branches(galhos) do arquivo, (A verde é a que estou usando no momento)
10 - git branch <nome da branch> << cria uma nova branch
11 = git checkout <nome da branch> << trocar a branch usada no momento para a branch nova
12 - git merge <nome da branch> << unir a branch do código para a branch atual
